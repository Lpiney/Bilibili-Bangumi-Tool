
# Bilibili-Bangumi-Tool

一个用于爬取和处理B站追番列表的工具，帮助用户快速获取和整理自己的追番信息。

## 📝 项目说明

**重要声明**：本项目所有代码均由 AI 生成，仅供学习和参考使用。

## 🚀 功能特点

- 🤖 **自动爬取**：使用 B 站 API 自动爬取指定用户的追番列表
- 📊 **详细信息**：获取番剧的标题、封面、评分、集数、发布时间等信息
- 🧹 **智能去重**：自动去除重复的番剧（如同时有一、二、三季的）
- 📱 **用户友好**：简洁的命令行界面，易于使用
- 📁 **数据导出**：将结果保存为 JSON 格式，方便后续使用

## 🔧 环境要求

- Python 3.7+
- 网络连接（能够访问 B 站）

## 📥 安装步骤

### 方法一：直接下载

1. 点击 GitHub 页面上的 "Code" 按钮，选择 "Download ZIP"
2. 解压下载的 ZIP 文件到任意目录
3. 进入解压后的目录

### 方法二：使用 Git 克隆

```bash
git clone https://github.com/Lpiney/Bilibili-Bangumi-Tool.git
cd Bilibili-Bangumi-Tool
```

### 安装依赖

在项目目录中运行以下命令：


## 💡 常见问题

### Q: 运行脚本时提示 "API 请求失败" 怎么办？

A: 可能是网络问题或 B 站 API 限制，请检查网络连接并稍后再试。

### Q: 爬取的追番数量与实际不符怎么办？

A: 可能是 B 站 API 返回的数据不完整，或者用户设置了隐私限制。

### Q: 处理后的列表中仍然有重复项怎么办？

A: 可能是去重规则不够完善，可以手动编辑 `bangumi_list_cleaned.json` 文件。
